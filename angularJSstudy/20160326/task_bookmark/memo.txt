JS
1. angular.module
    생성, 등록 및 앵귤러 모듈을 검색(retrieving)을 위한 전역 장소(global place). 사용 가능한 메커니즘을 등록함.

    // Create a new module
    var myModule = angular.module('myModule', []);

    // register a new service
    myModule.value('appName', 'MyCoolApp');

    // configure existing services inside initialization blocks. (블록 내부에 기존 서비스를 구성한다.)
    myModule.config(['$locationProvider', function($locationProvider) {
      // Configure existing providers
      $locationProvider.hashPrefix('!');
    }]);

    // load modules
    var injector = angular.injector(['ng', 'myModule'])

    // 사용법
    angular.module(name, [requires], [configFn]);

    //설명
    name : type은 string임. 생성할 이름이나 검색할 이름.
    ruquires(optional) : type은 !Array.<string>=  임. 지정하면 새 묘둘이 생선된다.
    configFN(optional) : type은 Function= 임.

ng-disable : 클릭 되고 ㅏㄴ되고
ng-if : true면 dom 생성, false면 dom 사라짐.
ng-show : dom은 남아있고, 디스플레이만 됐다가 안됐다가
ng-hide : show의 반대
